# Mercedes Landing Page

A responsive landing page for Mercedes vehicles built with HTML and CSS.

## Features

- Responsive design (desktop, tablet, mobile)
- Product grid with color selector
- Interactive carousel
- Store locator section
- Newsletter signup

## Technologies

- HTML5
- CSS3
- Google Fonts (Inter Tight, Roboto Serif)

## Setup

1. Clone the repository
2. Open `index.html` in your browser

## Structure

- `index.html` - Main HTML file
- `styles.css` - All styles and media queries
- `public/` - Images and assets
